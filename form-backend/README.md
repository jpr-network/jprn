# JPRN問い合わせフォーム送信設定

`contact.html`はGitHub Pages上で動作する独自デザインのフォームです。メール送信には、`Code.gs`をGoogle Apps Scriptのウェブアプリとして公開したURLを使用します。Google Formは使用しません。

## 設定手順

1. Google Apps Scriptで新規プロジェクトを作成します。
2. `Code.gs`の内容をプロジェクトへ貼り付けます。
3. 「デプロイ」から「新しいデプロイ」を選び、種類を「ウェブアプリ」にします。
4. 実行ユーザーを自分、アクセスできるユーザーを全員に設定してデプロイします。
5. 初回デプロイ時に、メール送信権限を承認します。
6. 発行された`/exec`で終わるURLを、`contact.html`の`jprn-form-endpoint`メタタグの`content`へ設定します。
7. 公開前にテスト送信し、入力者と管理者の両方へ同一内容が届くことを確認します。

公開ウェブアプリにはGoogle Apps Scriptのメール送信上限が適用されます。フォームには簡易的なスパム対策としてハニーポットと同一メールアドレスからの連続送信制限を実装しています。
